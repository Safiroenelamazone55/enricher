How the scoring works

①
Pattern generation

Up to 30 patterns per lead ranked by global frequency (firstname.lastname, flastname, etc.)

②
DNS MX validation

If the domain has no MX records, score drops to ≤5. Results cached 60 min.

③
SMTP + real send

All candidates probed in parallel. Real email sent via SES when SMTP is inconclusive.

④
Honest scoring

Verified = confirmed by server. Probable = best pattern match. Always shown with confidence level.
